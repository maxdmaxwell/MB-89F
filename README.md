## About MB 89F
A standard (algebraic) calculator that keeps every value as an exact fraction — you key a
calculation the normal way and press equals, and 1/3 + 1/6 stays exactly 1/2. Tape-measure
rounding (1/8–1/64), exact inch / foot / millimeter / meter conversion, mixed numbers,
memory, and a history tape. Works fully offline — no account, no ads, no tracking, and no
data collected.
